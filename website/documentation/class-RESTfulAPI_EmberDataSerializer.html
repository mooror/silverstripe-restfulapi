private static array $sideloaded_records

Sideloaded records settings Specify which relation's records will be added to the JSON root: 'RequestedClass' => array('RelationNameToSideLoad', 'Another')

Sideloaded records settings Specify which relation's records will be added to the JSON root: 'RequestedClass' => array('RelationNameToSideLoad', 'Another')

Non sideloaded response: { 'member': { 'name': 'John', 'favourites': [1, 2] } }

Response with sideloaded records: { 'member': { 'name': 'John', 'favourites': [1, 2] }, favourites': [{ 'id': 1, 'name': 'Mark' },{ 'id': 2, 'name': 'Maggie' }] }

Try not to use in conjunction with RESTfulAPI $embedded_records with the same settings.
